# Evento: DocumentReceived

## Tópico

document-received

## Publicador

CustomerDocumentValidation.Api

## Consumidor

CustomerDocumentValidation.Worker

## Quando é publicado

Após a API receber os dados cadastrais, salvar o documento no S3 e persistir o cadastro no MongoDB.
